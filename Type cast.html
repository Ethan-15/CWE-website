Type Casts

Type casts are to change a data type in python to a specific data type in order to perform the require operation by the user. Python uses type casts to determine the type of data that is going to be inputted.

Casting in python is done by either using Intergers, Strings or Floats.

  • Int: In python the variable “int” is used when using data like positive or negative numbers of any length. One thing to not however is that decimals cannot be used.

  • Str: In python the variable “str” is used when referring to the alphabet, using strings allow you to make use of words and sentences. One thing to take note of however is that strings in Python cannot be changed after they are created, meaning they are immutable.

  • Float: In python the variable “float” is a variable that converts a value into a floating point number. In simple terms it makes everything a decimal and is used when dealing with decimals inside of python. One thing to remember is that when using floats you aren’t able to use any letter of the alphabet.
