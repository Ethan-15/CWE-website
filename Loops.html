Loops

A loop in python is a control flow statement which is used to execute a group of statements as long as the conditions are satisfied. In simple words a loop works by executing a code if all the conditions to execute that code are true. Loops are used to repeat a certain code that is given to them.

For loop: A for loop in python is used when you have something that you want to repeat a specific amount of times. A for loop can be used to separate the items of a list.Here are some things to know:

  • A for loop will go through each item in a group.
  • You can choose what you want to say as the code lays out the items in your list using an indented block of code.
  • It stops when there are no more items.

While loop: While loops is used when you want to run a piece of code until a certain condition is met. When using while loops however there are certain rules that one must follow in order for their code to work which are:

  • The while loop evaluates the condition.
  • If the conditions are met then the code inside the while loop is executed.
  • The while loop will evaluate the condition again.
  • If the condition is false then the while loop stops.

Nested loop: A nested loop is a loop found within another loop. With nested loops you are able to run two different loops simultaneously. Here are some things to take note of when using nested loops:

  • Indentation (leaving space at the beginning of a sentence) matters as proper indentation is crucial in python.
  • Avoid too much nesting, this will make it difficult for you to understand.
  • It should be clear how both of the loops that you will be using interact with each other.
