Variables

A variable in python is a container that stores value. A variable is created when a value is assigned to it, thus anything can become a variable and the variable can later on be used down the line in other lines of code.

An example of the use of a variable in python is as a name to represent an object or something. Once the object is assigned a variable is can be referred to as that name. In python a variable is a container that stores values.

There are rules when using variables, these rules are:
  • Used small letter “p” when writing print or it will result in an error.

  • A python variable name cannot start with a number.

  • A python variable name can only contain numbers,letters and underscores.

  • Variables in python are case sensitive.( This means that NAME, and name are two different variables as they aren’t exactly the same)

  • Keywords cannot be used as the variable name in python.( Keywords are: and, or, not, if, elif, else, for, while, break, as, def, lamba, pass, return, True, False, try, with, assert, class, continue, del, except, finally, from, global, import, in, is, none, raise, yield, async, await)
